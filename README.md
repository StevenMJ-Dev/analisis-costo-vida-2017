# Análisis del Costo de Vida Mundial 2017

## Descripción

Análisis exploratorio de datos sobre el costo de vida de diferentes países del mundo durante el año 2017.

El proyecto permite comparar países mediante indicadores como el costo de vida, ranking global y ubicación por continente.

## Tecnologías utilizadas

- Python
- Pandas
- Matplotlib

## Análisis realizado

El proyecto incluye:

- Revisión inicial del dataset.
- Cantidad de filas y columnas.
- Cálculo del costo de vida promedio.
- Identificación del país con mayor costo de vida.
- Identificación del país con menor costo de vida.
- Análisis del costo de vida de Perú.
- Ranking global de Perú.

## Visualizaciones realizadas

- Top 10 países con mayor costo de vida.
- Top 10 países con menor costo de vida.
- Comparación del costo de vida de países de América.
